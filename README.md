# 🔮 ZodiNum Predictor 🔢  
### AI-Based Zodiac & Numerology Intelligence System  

---

## 📖 Overview  
ZodiNum Predictor is an AI-based system that combines **zodiac astrology** and **numerology** with **Natural Language Processing (NLP)** to generate personalized predictions.  

The system provides users with:  
- AI-generated horoscope predictions  
- Numerology insights (Mulank, Bhagyank, Kua)  
- Personality traits  
- Career suggestions  
- Love compatibility  

---

## 🚀 Features  
- 🔮 AI-generated horoscope (General, Love, Career)  
- 🔢 Numerology calculations:  
  - Mulank (Birth Number)  
  - Bhagyank (Life Path Number)  
  - Kua Number  
- 🧠 Personality insights  
- 💼 Career recommendations  
- ❤️ Love compatibility analysis  
- 🍀 Lucky number, color, and fortune level  
- 🤖 NLP-based text generation (Markov Chain)  

---

## 🧠 Technologies Used  
- 🐍 Python  
- 🤖 Natural Language Processing (NLP)  
- 🔗 Markov Chain Model  
- 📊 Rule-Based AI System  

---

## ⚙️ How It Works  
1. 👤 User enters:  
   - Zodiac sign  
   - Date of Birth (DDMMYYYY)  
   - Gender  

2. ⚡ System processes:  
   - Generates horoscope using NLP  
   - Calculates numerology values  
   - Matches personality profiles  

3. 📤 Outputs:  
   - Horoscope predictions  
   - Numerology insights  
   - Personality, career, and love suggestions
